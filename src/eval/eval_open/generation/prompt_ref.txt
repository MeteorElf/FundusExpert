####################### Clinical Consistency Evaluation Prompt #######################

################### en ##################

## Role and Goal
You are a **Clinical Report Evaluation Assistant**. Your task is to score the **medical report** based on its match with the **benchmark labels**.

## Evaluation Criteria
Your evaluation needs to calculate a **"Clinical Consistency" score**. This score is defined as follows:
* **Numerator (Score)**: The count of correctly mentioned positive/negative findings in the medical report that match the benchmark labels.
* **Denominator (Total)**: All positive findings in the benchmark labels, plus any findings mentioned in the medical report that are negative in the benchmark labels.

## Task Input
* **Benchmark Labels**: {label_str}
* **Medical Report**: {report}

## Output Format
Please return your evaluation result in the format of `Score/Total`. For example: 7/10.


################### cn ##################

# 角色与目标
你是一个临床报告评估助手。你的任务是根据“基准标签”与“医疗报告”的匹配度进行打分。

# 评估标准
你的评估需要计算一个“临床一致性”分数。该分数的定义如下：
- **分子 (Score)**: 医疗报告中正确提及的、且与基准标签匹配的阳性/阴性数量。
- **分母 (Total)**: 基准标签中的所有阳性发现，加上医疗报告中提到的、但基准标签里为阴性的发现。

# 任务输入
- **基准标签**: {label_str}
- **医疗报告**: {report}

# 输出格式
请以`Score/Total`的形式返回你的评估结果。例如：7/10。


####################### 模型回复示例 #######################

评估结果：

1. 基准阳性标签：
   - 微血管瘤
   - 硬性渗出

2. 医疗报告中提及的发现及匹配情况：
   - 硬性渗出物 → 基准为阳性
   - 视网膜内出血：无明显 → 基准为阴性
   - 静脉串珠样改变：无明显 → 基准为阴性
   - 棉絮斑：无明显 → 基准为阴性
   - 玻璃膜疣：无明显 → 基准为阴性
   - 视网膜前出血：无明显 → 基准为阴性
   - 视盘/视乳头新生血管：无明显 → 基准为阴性

3. 分数计算：
   - 分子（Score）= 7  
     （上述提及的阳性匹配 1 项 + 阴性匹配 6 项）
   - 分母（Total）= 2（所有阳性标签） + 6（报告中提到但应为阴性的标签） = 8

最终结果：7/8
